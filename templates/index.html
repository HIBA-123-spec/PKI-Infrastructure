📋 Guide de démarrage rapide

Suivez ces étapes dans l'ordre pour construire votre PKI complète :

1️⃣
Créer la Root CA
Onglet "Root CA" → Remplir le formulaire → Générer
2️⃣
Créer l'Intermediate CA
Onglet "Intermediate CA" → Remplir le formulaire → Générer et signer
3️⃣
Émettre des certificats feuilles
Onglet "Émettre un certificat" → Remplir → Signer par Intermediate CA
